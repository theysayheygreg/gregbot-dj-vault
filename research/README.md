# Research Workspace

This directory holds the reverse-engineering and artifact-analysis side of DJ Vault.

## Structure

- `analysis/`
  - durable Markdown writeups
- `corpus/`
  - extracted artifact notes and local working material
- `manifests/`
  - machine-readable inventories and source maps
- `sources/`
  - notes on official vendor surfaces and acquisition strategy

## Rules

- Keep claims tied to evidence.
- Prefer official public sources.
- Note missing artifacts explicitly.
- Keep downloaded binaries out of git unless there is a specific reason not to.
